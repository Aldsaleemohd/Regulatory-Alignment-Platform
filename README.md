# Regulatory Alignment Platform | منصة التوافق التشريعي

A Legal Compliance & Risk Analysis platform designed for government and regulated organizations.

## Features

- **Dashboard** — Overview of compliance metrics, risk distribution, and alerts
- **Regulations Library** — Manage laws, policies, and circulars with clause-level detail
- **Documents Library** — Store and manage contracts, agreements, and memos
- **Compliance Check Engine** — Analyze documents against regulations with keyword-based clause matching, weighted scoring, and risk assessment
- **Conflict Analysis** — Identify which clauses conflict with which regulations
- **Reports** — Generate structured compliance reports
- **Bilingual** — Full Arabic (RTL) and English (LTR) support
- **Role-Based UI** — Admin, Legal, and Viewer roles

## How to Run

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).
